# CTF Platform

REST API for Capture The Flag hacking events, built with Spring, provides JWT authentication, MySQL database


# Endpoints

## Authentication
- `POST /authenticate`

### Challenges
- `GET /challenges`
- `GET /challenges/id`
- `POST /challenges`
- `PUT /challenges/id`
- `DELETE /challenges/id`

### Users
- `GET /users`
- `GET /users/id`
- `POST /users`
- `PUT /users/id`
- `DELETE /users/id`

### Flag validation
- `POST /validate`


### Ranking
- `GET /ranking`
